# Quick Start - Super Simple!

## 🚀 Launch the Crypto Dashboard

From anywhere in this repository, just run:

```bash
./podserve
```

That's it! Opens at: http://localhost:9000/demo.html

## 🏢 Launch the VC Dashboard

```bash
./podserve-vc
```

Opens at: http://localhost:8000/demo.html

## 🎯 What You Get

### Crypto Version (`./podserve`)
- **Port**: 9000
- **Topics**: RWAs, ETH Restaking, Bitcoin L2s, Memecoins
- **Personalities**: Vitalik, Raoul Pal, Larry Fink
- **Podcasts**: Bankless, What Bitcoin Did, The Defiant

### VC Version (`./podserve-vc`)
- **Port**: 8000
- **Topics**: AI Infrastructure, B2B SaaS, Developer Tools
- **Personalities**: Brad Gerstner, Harry Stebbings
- **Podcasts**: 20VC, All-In, Stratechery

## Running Both

Open two terminals:
```bash
# Terminal 1
./podserve-vc

# Terminal 2
./podserve
```

Now both are running simultaneously!
