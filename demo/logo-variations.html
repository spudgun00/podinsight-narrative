Recommendation

Option 1 (Outlined Circles) offers the cleanest solution - it maintains the synthesis concept through elegant overlaps without any color mixing issues. The 2px stroke weight provides good visibility at all sizes while keeping the design sophisticated and modern.

Option 4 (Outlined + Light Fill) is a close second if you want slightly more visual weight while still avoiding the opacity issues.
