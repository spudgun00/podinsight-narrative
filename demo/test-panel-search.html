Search Panel Input Test

This test verifies that the search input has been successfully added to the search panel header.

Test Steps:

  1. Type "vertical AI" in the main search bar and press Enter
  2. The search panel should open from the right
  3. Check that the panel header shows:
    • "Search: vertical AI" as the title
    • A search input field below the title (pre-filled with "vertical AI")
    • A search button next to the input
  4. Try typing a new search term (e.g., "Series A valuations") in the panel input
  5. Press Enter or click the Search button
  6. Check the console for: "New search from panel: Series A valuations"
  7. The title should update to "Search: Series A valuations"
  8. Verify that the "New Search" button at the bottom has been removed

Expected Behavior:

  • ✅ Search input is visible in the panel header
  • ✅ Input is pre-filled with current search query
  • ✅ Enter key triggers search
  • ✅ Search button works
  • ✅ Title updates with new search
  • ✅ Console logs new searches
  • ✅ No "New Search" button at bottom
